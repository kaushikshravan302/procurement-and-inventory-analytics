# Procurement Operations Dashboard – Power BI

## Project Overview
This project focuses on procurement and operational analytics using Power BI. The dashboard analyzes procurement status, vendor activity, RM/PM material categorization, delivery trends, and operational bottlenecks.

---

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

---

## Features
- Procurement KPI tracking
- Vendor analysis
- RM vs PM procurement comparison
- Priority-based procurement monitoring
- Purchase status tracking
- Trend analysis
- Operational insights section

---

## Data Cleaning & Transformation
- Managed inconsistent procurement records
- Handled unknown category values
- Validated purchase workflow data
- Standardized supplier information
- Created semantic relationships

---

## Key Insights
- RM procurement contributes majority of purchase value.
- Under-process purchases dominate procurement workflow.
- Critical procurement records require operational attention.
- Procurement trends fluctuate significantly across delivery dates.
